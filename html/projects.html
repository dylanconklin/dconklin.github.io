Adjacency Table
Swift, SwiftUI
Adjacency Table is a program that generates an adjacency table based on data. The data contains a name to indicate any given point on a map, another name to indicate another point on a map, and a distance between the two points.

I originally wrote this program for a class, then continued developing it into an app.
