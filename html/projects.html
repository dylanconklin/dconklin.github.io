MESSenger

MESSenger is an instant messaging server that you can connect to via telnet. It features an easy to use interface with a help option, utilizes a SQLite database to store messages locally, and provides public and private messaging options.

Eldrow (Wordle)

I recently posted a program on my Gitlab called Eldrow, which is a command line Wordle clone. Included is a program that will generate a dictionary for the game, and a sample dictionary file to generate from.

I plan on adding more features to this game, such as a solver, arrow key support, and will make the current features more rebust.

Adjacency Table

Adjacency Table is a program that imports data from a file, and generates an adjacency table based on the data. The data file contains a number to indicate any given point on a map, another point to indicate another point on a map, and a distance between the two points.

I originally wrote this program for a Discrete Structures class, and reused the code later for an Algorithms class, and fixed it up with the help of two of my lab partners. We were pleasantly surprised with the resulting program.

Number Converter

I recently posted a program on my Gitlab called "Number Converter". It allows you to put in a value and change the base of that number, and prints out the number in the new base.

It's a little limited in scope, because it can't handle negative numbers, but is very scalable, and allows you to use whatever character set you want.
